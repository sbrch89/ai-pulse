# AI Pulse Project

## خطوات التشغيل:

1. انسخ `.env.local` وأضف فيه مفتاح OpenAI API الخاص بك.
2. ثبت الحزم:
```
npm install
```
3. شغل المشروع محلياً:
```
npm run dev
```
4. افتح المتصفح على:
```
http://localhost:3000
```
## رفع على Vercel

- اربط المستودع بـ Vercel.
- تأكد من إضافة متغير البيئة `OPENAI_API_KEY` في إعدادات المشروع على Vercel.
- اضغط Deploy.
